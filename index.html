Urban Prism

Urban Prism is an AI-assisted grievance intelligence portal designed to transform citizen complaints into meaningful urban insights. The platform leverages multilingual AI intake, geospatial clustering, and asset mapping to analyze complaint patterns across locations. By incorporating risk-based prioritization, the system helps authorities identify critical urban issues faster and support data-driven city management decisions.

Market Mate

MarketMate is a MERN-based demo platform that enables local shopkeepers to list products and allows users to compare prices across nearby stores. The system features shopkeeper onboarding with admin approval workflows, product management dashboards, and public product browsing with search and filtering capabilities. A clean dark/light theme toggle enhances usability, making the platform suitable for showcasing full-stack development workflows.

LUNEX

LUNEX is a full-stack smart laundry management system developed to streamline washing machine usage in hostel environments. The platform combines web-based slot booking with RFID and ESP32-based hardware integration to enable secure machine access and real-time usage control. LUNEX demonstrates the integration of IoT hardware with software systems to improve resource utilization and reduce conflicts in shared facilities.

React Node.js Express MongoDB
