# Jayasuriya S - Personal Portfolio

A modern, responsive personal portfolio website built using HTML, CSS, and JavaScript.

## Folder Structure

```text
Portfolio/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── assets/
│   ├── images/
│   ├── videos/
│   └── pdfs/
├── documents/
├── Havox/
└── Video and pics/
```

## Run Locally

1. Open the `Portfolio` folder in VS Code.
2. Run with the Live Server extension, or open `index.html` directly in a browser.

## Notes

- Resume button directly downloads the PDF from `assets/pdfs/resume-jayasuriya.pdf`.
- Contact email, LinkedIn, GitHub, SkillRack, and LeetCode links are already configured.
- Images use lazy loading and async decoding; videos use deferred preload for better mobile performance.
